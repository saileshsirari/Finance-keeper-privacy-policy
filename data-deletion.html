Data Deletion Policy

Finance Keeper

Overview

Finance Keeper does not create user accounts and does not store financial data on external servers. All transaction data is stored locally on your device.

1. Deleting Local App Data

To permanently delete all your financial data, SMS processing data, and notes:

Option 1: Uninstall the app from your device.
This removes all locally stored application data.

Option 2 (If Local Backup Was Created):
If you manually created a backup/export file, you must delete that file separately using your device’s file manager.

Finance Keeper does not retain any financial records on external servers. All transaction data remains on your device unless you manually export it.

2. Analytics & Crash Reporting Data

The App uses:

  • Firebase Analytics
  • Firebase Crashlytics

These services may collect limited anonymous technical data such as:

  • App interaction events
  • Crash logs
  • Diagnostics information
  • Device or installation identifiers (e.g., Firebase Installation ID)

No SMS content, financial transaction details, or backup files are transmitted to Firebase or any external server.

Data sent to Firebase is transmitted using encrypted connections (HTTPS).

Uninstalling the App stops further analytics collection and resets the Firebase Installation ID associated with your device.
